HTML

  • meta data in the head element tells a computer what is expected to be read from the html code
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • use H1-6 to change how bold you want your headers to be
  • The charset="utf-8" is meta data that denotes the type of characters that will be used

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Attributes can be assigned to multiple classes by assigning them in the CSS code
  • CSS gives color and demension to the HTML code

Git

  • The ls command tells you all files within a directory
  • git bash is a tool to control versions of the product or project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "" commits changes you have made to files within directories after using git add -A

JavaScript

  • scripts are exectued top to bottom in order
  • for loops can be used to preform a task in a predictable pattern
  • if statements can be useful to execute code in a particular order
  • script src="./assets/script.js is how we connect script.js to the HTML code
  • We can use the console.log method to output a message to the web console by adding an argument
